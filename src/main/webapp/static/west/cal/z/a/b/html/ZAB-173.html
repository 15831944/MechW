疲劳分析的容器总结

  1. 分析设计的压力容器的设计文件，除进行设计、校核、审核3级签署外，还应当由压力容器设计单位技术负责人或其授权人批准(4级签署)。 [TSG 21-2016 p15 3.1.4.2]
  2. 设计图样中应注明设计使用年限(寿命)及允许的交变载荷循环次数。 [TSG 21-2016 p15 3.1.4.4.1 (6)]
  3. 接管(凸缘)与壳体之间的焊接接头以及夹套容器的焊接接头，应当采用全焊透结构。 [TSG 21-2016 p21 3.2.2.2]
  4. 按照分析设计标准建造的压力容器，其A、B类对接接头应进行100%无损检测(RT/UT/TOFD)。 [TSG 21-2016 p23 3.2.10.2.2.2]
  5. 按照疲劳分析设计的容器，应当去除纵、环焊缝的余高，使焊缝表面与母材表面平齐。 [TSG 21-2016 p38 4.2.4.2]
  6. 操作过程中有振动、较大的温度波动时，不得采用强度胀接。 [GB/T 151-2014 p25 6.6.1.1]
  7. 操作过程中有较大振动时，管板不得采用强度焊接。 [GB/T 151-2014 p27 6.6.2.1]
  8. 有振动或循环载荷时，管板应采用胀焊并用。 [GB/T 151-2014 p28 6.6.3.1]
  9. 不得采用补强圈结构。 [JB/T 4736-2002 p9 1.4]
