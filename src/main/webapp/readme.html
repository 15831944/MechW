一、网站访问

  • 推荐公网访问，网络延迟过高时，会影响界面操作体验(有卡顿感)

二、数据输入

  • 要选择材料，必须先输入设计温度，程序会根据设计温度自动筛选适用的材料
  • ASME VIII-1 计算程序的Material Name选择框，大约有1700-1800种材料，靠下拉滚动选择极不方便，可以直接键入关键字筛选(大小写敏感)，类似搜索引擎的自动智能提示
  • 程序会实时检查输入数据，有错误会在页面下方红字提示，正确则无提示

三、计算结果

  • 数据输入完整后，计算结果会实时在页面下方显示，红色表示不合格项，无论结果合格与否，计算书都可以下载
  • 输入数据修改后，计算结果会实时更新，并在页面下方显示
